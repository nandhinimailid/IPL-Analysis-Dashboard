# Project Purpose

This dashboard was built to serve as a complete analytical solution for the Indian Premier League (IPL), covering data from the 2008â€“2022 seasons. Its primary goal is to translate raw match, player, and season-level data into meaningful insights that can support informed decision-making for stakeholders such as the BCCI, franchise owners, coaching staff, analysts, and even general cricket fans.

The report showcases season-specific insights, highlighting key outcomes like the championship team, Orange Cap and Purple Cap winners, and overall boundary counts. It also includes in-depth player-level analysis, enabling users to explore batting and bowling performance for individual athletes across multiple seasons. Beyond player metrics, the dashboard visualizes broader trends such as toss influence on match results, team win distributions across venues, and season-by-season team performance.

Overall, this project demonstrates how Power BI can be used to address client problem statements by combining data modeling, DAX calculations, and interactive visuals to provide a clear, data-driven picture of IPL history.
